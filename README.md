Project Description

This project aims to put into practice what has been learned about the UNIX/Linux operating system, the use of Git as a version control tool, and the use of digital signatures with GPG. Through this work, we intend to understand how to ensure the authenticity of changes made to a repository and improve the use of fundamental tools in the technological field.

Objectives
Understand in a practical way how the UNIX/Linux operating system works.
Use basic commands from the terminal to perform different tasks.
Learn to use Git as a version control system.
Apply digital signatures with GPG to validate the authorship of commits.
Manage repositories on platforms such as GitHub or Codeberg in an organized manner.
